Upcoming Names

GET https://names.sparky.rip/upcoming?three_letter=<true/false>

Returns the next 50 upcoming names. Each name will be returned along with it's searches (pulled from NameMC), it's unixDropTime, and it's utcDropTime.

If three_letter is set to true only three letter names will be returned.

Name Droptime

GET https://names.sparky.rip/droptime/<name>

Returns the droptime of a specified name. Included with name will be unixDropTime and utcDropTime. Only names that are above 3 searches and are dropping in the next 24 hours can be returned.

Response

{
    "name": "657",
    "unixDropTime": "1623236193",
    "utcDropTime": "2021-06-09T10:56:33.000Z"
}

Error Response

If the droptime of the specified name cannot be found, the following error will be returned: 
{
    "error": "INVALID_NAME"
}
