Upcoming Names

GET https://names.sparky.rip/upcoming?three_letter=<true/false>

Returns the 50 next upcoming names. Each name will be returned along with it's searches, it's unixDropTime, and it's utcDropTime.

If three_letter is set to true only three letter names will be returned.
